# Multi-Agent Research Team

This repo demonstrates a LangGraph-based research assistant system that uses multiple specialized agents to retrieve, summarize, and synthesize information from various sources. Designed for academic and financial domains.

## Features

- LangGraph multi-agent coordination
- Researcher, summarizer, synthesizer roles
- Document-based and web-based data sources
- Uses LangChain for retrieval and QA
- Handles long-context PDFs and web results

## Technologies

- LangGraph
- LangChain
- OpenAI / Claude (LLMs)
- FAISS or Pinecone for vector retrieval

## Use Cases

- Topic briefings for finance or healthcare
- Automated academic literature reviews
- Assistant for analysts or research teams
